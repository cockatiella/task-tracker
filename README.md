# task-tracker

> A Vue.js + Vite project

This is a simple Vue.js project that I created while learning Vue.js. It is a task tracker that allows you to add tasks, delete tasks, and toggle the reminder on and off. 

## Project setup
```
npm install
npm run dev
```
